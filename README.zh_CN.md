# FoloToy 服务部署说明

一个可定制的，自托管的服务器，用于您购买或者 DIY 的 FoloToys。

- 语音输入和语音输出
- LLM（大型语言模型）支持OpenAI/Azure OpenAI
- SST（声音转文本）支持OpenAI Whisper/Azure Speech Service
- TTS（文本转声音）支持OpenAI TTS/Azure Speech Service/Elevenlabs/Edge TTS
- AI 对话数据发布到**MQTT**服务器
